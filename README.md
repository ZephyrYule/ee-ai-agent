# ee-ai-agent
# 电子信息工程 + AI Agent 50天实战项目
深圳大学电子信息工程专业 | 通信+AI 差异化能力打造

## 项目简介
本仓库为50天零基础AI Agent开发学习计划的完整作品集，所有项目100%紧扣电子信息工程核心场景，打造「电子信息+AI」的复合型竞争力，适配考研复试与校招/实习求职。

## 项目列表
### 项目一：电子硬件故障智能诊断Agent
- 核心功能：输入硬件故障现象/测试数据，自动输出故障分类、排查步骤、解决方案
- 核心技术：Prompt工程、大模型API调用、Gradio可视化
- 应用场景：模电/数电实验故障排查、硬件调试、电路排障

### 项目二：数字信号处理（DSP）智能分析与代码生成Agent
- 核心功能：输入信号需求/异常数据，自动完成信号分析、异常定位、可运行DSP代码生成
- 核心技术：Function Calling工具调用、信号预处理、多平台代码生成
- 应用场景：DSP课程设计、单片机/FPGA开发、信号处理实验

### 项目三：电子信息专业核心规范智能问答Agent
- 核心功能：基于3GPP协议、芯片手册、总线规范等专业知识库，实现精准无幻觉问答
- 核心技术：RAG检索增强生成、向量检索、大模型API调用
- 应用场景：专业知识查询、考研复试备考、协议/规范查阅
